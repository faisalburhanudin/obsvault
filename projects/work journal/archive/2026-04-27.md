 ### Backfill flow (current)

 1. PoolFleet.startup() or post-assignment start_app() calls _backfill_pool().
 2. _backfill_pool():
     - runs _garbage_collect()
     - computes deficit: POOL_SIZE - unassigned_apps
     - reserves staged rows
     - schedules AppLifecycle.provision_one() tasks.
 3. AppLifecycle.provision_one():
     - transition staged -> creating
     - fleet.start_app(internal_browser_id) (creates Fly app/machine)
     - builds predefined US proxy_config
     - calls fleet.query_app(..., run_configure=proxy_config is not None, proxy_config=proxy_config)
     - transition creating -> standby
 4. DirectFleet.query_app() forwards proxy_config into FlyApp.configure(proxy_config=...).
 5. FlyApp.configure() forwards to FlyMachine.configure(proxy_config=...).
 6. FlyMachine.configure():
     - if proxy_config provided: use it
     - else fallback to get_proxy_config_from_headers(settings) (request path)
     - applies proxy + validates egress IP change.
