# Personal Planning System

A lightweight, flexible planning system designed that adaptable.

## Philosophy

- **Tiny habits over big goals** — Sustainable progress beats burnout
- **Flexible timelines** — Adjust based on reality, not rigid schedules
- **Family first** — Planning serves life, not the other way around
- **Compounding** — Small daily actions add up over time

## File Structure

```
planning/
├── README.md        # This guide
├── Yearly.md        # Big picture & quarterly themes
├── Monthly.md       # Month-level goals & review
└── Weekly.md        # Weekly tasks & immediate focus
```

## How to Use

### Yearly.md

**Review:** Quarterly (or when life changes)  
**Purpose:** Set themes and boundaries for the year

- Don't overload — this is direction, not a to-do list
- Adjust quarters based on actual experience
- 2025 is designed for "survival + exploration" with a newborn

### Monthly.md

**Review:** End of each month  
**Purpose:** Bridge yearly themes to weekly actions

- Keep goals realistic for your life stage
- Use "Notes" section to capture learnings
- It's okay to have empty months — that's data too

### Weekly.md

**Review:** End of each week  
**Purpose:** Immediate action items

- 3-5 goals max — constraint forces priority
- Use "Review" questions to iterate
- Archive old weeks at bottom or in git history

## Workflow

1. **Start of week:** Check Weekly.md, pick top 3 priorities
2. **During week:** Tick off items, capture ideas in notes
3. **End of week:** Answer review questions, carry over unfinished items if still relevant
4. **End of month:** Review Monthly.md, check against Yearly themes
5. **Quarterly:** Revisit Yearly.md, adjust upcoming quarters

## Tips

- **Empty checkboxes are information** — not failure
- **Carry over sparingly** — if it didn't happen, maybe it's not a priority
- **Add context** — use notes to remember _why_ a goal matters
